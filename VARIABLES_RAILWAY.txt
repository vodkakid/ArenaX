  ArenaX Bot v5 — Variables de entorno para Railway
  Ve a: tu proyecto → Variables → agrega cada una
═══════════════════════════════════════════════════════

── OBLIGATORIAS ────────────────────────────────────────

BOT_TOKEN
  Token de @BotFather
  Ejemplo: 7123456789:AABBCCDDEEFFaabbccddeeff

ADMIN_ID
  Tu Telegram User ID (número entero)
  Obtén escribiéndole a @userinfobot
  Ejemplo: 123456789

ADMIN_CHANNEL_ID
  ID de tu canal privado de notificaciones admin
  El bot debe ser administrador del canal
  Ejemplo: -1001234567890

GROUP_ID
  ID del grupo oficial ArenaX
  Ejemplo: -1001234567890

TOPIC_ANNOUNCEMENTS_ID = 2
TOPIC_MATCHMAKING_ID   = 32
TOPIC_RESULTS_ID       = 14

── DATOS DE PAGO MÓVIL ARENAX ──────────────────────────

ARENAX_BANK    → Ejemplo: Banesco (0134)
ARENAX_PHONE   → Ejemplo: 0412-1234567
ARENAX_CEDULA  → Ejemplo: V-12345678
ARENAX_NAME    → Ejemplo: ArenaX Games

── MODELO ECONÓMICO (opcionales, ya tienen valores) ────

ENTRY_FEE_USD   = 1.50   (inscripción por partida)
WIN_PRIZE_USD   = 1.00   (premio neto al ganador)
MIN_WITHDRAW_USD= 2.50   (mínimo de retiro)

── OPCIONAL: GOOGLE SHEETS ─────────────────────────────

SHEETS_SPREADSHEET_ID
  ID de tu Google Sheet

GOOGLE_CREDENTIALS_JSON
  Contenido completo del credentials.json como texto
